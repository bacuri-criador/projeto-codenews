Política de Privacidade

Última atualização: 15/01/2025

1. Introdução

Esta Política de Privacidade tem como objetivo informar de forma transparente como são coletados, utilizados e armazenados os dados pessoais dos usuários do Sistema de Gerenciamento da UBS - CodeNews, em conformidade com a LGPD.


2. Dados coletados

Coletamos apenas os dados necessários ao atendimento: nome, CPF, data de nascimento, contatos e registros médicos quando fornecidos para fins clínicos.


3. Finalidade

Os dados são utilizados para identificar pacientes, registrar atendimentos, e cumprir obrigações legais.


4. Retenção

Dados serão mantidos por até 2 (dois) anos após o último atendimento ou conforme exigência legal.


5. Contato

Assuntos relativos à privacidade podem ser encaminhados para: suportecns@gmail.com
