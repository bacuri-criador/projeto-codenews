POLÍTICA DE PRIVACIDADE

Proteção e uso responsável dos dados dos pacientes

Última atualização: 15/01/2025

1. Introdução

Esta Política de Privacidade tem como objetivo informar de forma transparente como são coletados, utilizados e armazenados os dados pessoais dos usuários do Sistema de Gerenciamento da UBS - Code News, em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018 – LGPD) e demais legislações aplicáveis.

2. Dados coletados

O sistema coleta apenas os dados estritamente necessários para o funcionamento e identificação dos usuários:

  • Nome completo
  • CPF
  • Data de nascimento

Esses dados são utilizados exclusivamente para identificação do paciente e gerenciamento de atendimentos dentro da UBS.

3. Finalidade do uso dos dados

Os dados pessoais são utilizados para:

  • Identificar o paciente no sistema;
  • Registrar e acompanhar atendimentos e procedimentos realizados;
  • Garantir a segurança e a integridade das informações cadastrais;
  • Cumprir obrigações legais e regulatórias dos serviços de saúde.

4. Armazenamento e período de retenção

Os dados coletados serão armazenados com segurança e confidencialidade, sendo mantidos por um período máximo de 2 (dois) anos a partir da data do último atendimento.

Após esse período, as informações serão anonimizadas ou excluídas de forma segura, conforme determina a LGPD.

5. Compartilhamento de dados

Os dados pessoais não serão compartilhados com terceiros, exceto em casos:

  • De exigência legal ou judicial;
  • De cumprimento de obrigações junto a órgãos públicos de saúde, quando previsto em lei;
  • De necessidade técnica para manutenção e segurança do sistema (prestadores de serviço sob contrato de confidencialidade).

6. Direitos do titular dos dados

Nos termos da LGPD, o titular dos dados tem direito a:

  • Confirmar a existência de tratamento de dados;
  • Acessar, corrigir ou atualizar suas informações;
  • Solicitar anonimização, bloqueio ou eliminação de dados;
  • Revogar o consentimento a qualquer momento.

Os pedidos podem ser feitos pelo e-mail: suportecns@gmail.com.

7. Segurança da informação

O sistema utiliza medidas técnicas e administrativas adequadas para proteger os dados pessoais contra acessos não autorizados, perda, alteração ou destruição.

O acesso às informações é restrito apenas a profissionais devidamente autorizados.

8. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos a leitura regular deste documento para se manter informado sobre eventuais mudanças.
